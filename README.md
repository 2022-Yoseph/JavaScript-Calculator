# 🧮 JavaScript Calculator

A fully functional calculator built with React that performs basic arithmetic operations. It adheres to the requirements of the freeCodeCamp Front End Libraries Certification.

---

## 🚀 Features

- ➕ Addition, ➖ Subtraction, ✖️ Multiplication, ➗ Division
- 🟰 Equals button for evaluation
- 🔢 Real-time expression display
- 🧹 Clear all (`AC`) and delete last entry (`←`)
- 🧠 Handles multiple operations and decimal points
- ❌ Prevents multiple operator misuse (e.g., `++`, `--`)
- 📱 Responsive and modern UI

---

## 🛠️ Technologies Used

- **React.js**
- HTML5 + CSS3
- JavaScript (ES6)
- Basic math parsing with `eval()` (or alternatives like `math.js` for security)

---

## 📦 How to Run

1. Clone this repository:

```bash
git clone https://github.com/your-username/javascript-calculator.git
cd javascript-calculator
````

2. Open `index.html` in a browser **OR** use a live server (like Live Server in VS Code).

---

## ✅ User Stories (freeCodeCamp Spec)

* ✅ I can see a display for the current expression and result
* ✅ I can input numbers and operators
* ✅ I can calculate a result by pressing `=`
* ✅ I can clear the input with the `AC` button
* ✅ I can handle floating point numbers
* ✅ No invalid operator combinations (`++`, `..`, etc.)
